- We created a javascript application uses differents technologies and libraries detailed what follows. The application is wrapped in an HTML page and loaded by a web browser. 
The application is divided in 3 tabs, on for each map we decided to represent.
In function of which one is selected, the application loads the appropriate data. This way, we try to decrease the impact of the huge amount of data that the web browser has to load at the same time.

- LeafletJS : We used the library LeafletJS to have interactive maps. This system allows to add different kinds of layer that we defined by ourself. The main advantage is that these layers can be superposed and are over the world map layer.
